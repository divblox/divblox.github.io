## What is divbloxPHP?

divbloxPHP is a full stack web and mobile app development framework that allows anyone, from designers and analysts, to hard-core developers, to collaborate and create amazing experiences in no time.
With divbloxPHP you can build progressive web apps that can also seamlessly be converted to native mobile apps, all with ONE code base.
